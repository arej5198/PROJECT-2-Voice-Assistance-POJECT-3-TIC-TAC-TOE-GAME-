Project 2: Voice Assistant

The Voice Assistant project involves creating a simple virtual assistant using Python.
The assistant can listen to voice commands, convert speech into text, and perform tasks such as telling the time or opening websites.
This project uses libraries like speech recognition and text-to-speech to simulate real-world assistants like Siri or Alexa.
It helps in learning automation, working with external libraries, and building interactive applications.

Project 3: Tic-Tac-Toe Game

The Tic-Tac-Toe project is a simple game developed in Python where a user can play against the computer.
The program uses basic game logic and the random library to generate the computer’s moves.
It checks for winning conditions and declares the result.
This project helps in understanding conditional statements, lists, loops, and game development fundamentals.
